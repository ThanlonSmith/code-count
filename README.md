>**我的上线作品：[39毕设网[https://www.bysj39.com]](https://bysj39.com)**
#### 1. 系统需求
简而言之，就是工程师(用户)上传代码文件，系统计算代码行数并使用图表把代码提交记录等相关信息可视化地展示给用户，用户可根据图表直观地看到自己的每天写了多少行的代码。本项目开源，并在寻找合作伙伴。
#### 2.  采用的技术/语言
概括如下：Python3+Flask+Nginx+uWSGI+Git
#### 3. 系统功能简介
##### 3.1 用户注册
##### 3.2 用户登录
##### 3.3 用户信息管理
##### 3.4 代码记录提交
##### 3.5 代码记录管理
